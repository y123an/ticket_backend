# ticket_backend

## General Info
- its a backend of a ticket booking web app for movies, sports, concerts and other special events

## Technologies
```bash
- Node.js
- MongoDB
```
# Get started
First, to install the packages.run 
```bash
- npm install
```
to run the server
```bash
- nodemon index.js
```

## Features

- accepts an api request for and a response with a a ticket info
